# ChristianASC5 Claude Code Marketplace

Personal marketplace for Claude Code plugins.

## Install

```bash
/plugin marketplace add ChristianASC5/christianASC5-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [llm-wiki](./plugins/llm-wiki) | Persistent, compounding knowledge base — Karpathy's LLM Wiki pattern for Claude Code |

## Usage

After adding the marketplace, install individual plugins:

```bash
/plugin llm-wiki@christianASC5-marketplace
```
